MIP-MLP is preprocessing tool for 16S values.

The input file should contain detailed unormalized OTU/Feature values, and the appopriate taxonomy as a csv file,

and a possible tag file, with the class of each sample. See Example page for the formal of the input files. Note that the OTU file should have a taxonomy column in it.

The tag file is not used for the preprocessing, but is used to provide some statistics on the relation between the features and the class.

One can run MIP-MLP even without a tag file.
